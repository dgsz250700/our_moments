# Our Journal

A romantic digital scrapbook designed to collect and revisit shared memories in a warm, elegant, and interactive way.

## Overview

**Our Journal** is a memory-based web app where special moments can be saved, organized by date, and revisited through a soft scrapbook-style interface. The project was designed to feel personal and intimate, combining a calendar view with visual memory cards and location-based features.

The goal is to create a digital space that feels more emotional than a traditional gallery or notes app — a place to preserve meaningful memories in a simple and beautiful format.

## Features

- **Calendar-based memory navigation**
  - Browse memories by day and month
  - Highlight dates with saved entries

- **Memory cards**
  - View memories with photos, titles, descriptions, dates, and locations
  - Clean, journal-inspired layout for storytelling

- **Add new memories**
  - Create and save new moments directly from the interface

- **Places section**
  - Dedicated area for meaningful shared places
  - Currently in development

- **Warm visual design**
  - Soft neutral palette
  - Romantic scrapbook/journal aesthetic
  - Minimal and emotional user experience

## Tech Stack

- React / Expo
- JavaScript / TypeScript
- CSS / custom styling
- Frontend-focused prototype build

## Project Status

This project is currently a **work in progress**.  
Core memory browsing and journal-style display are implemented, while some features such as **Our Places** are still under development.

## Why I Built It

I wanted to build a project that felt both technically interesting and emotionally meaningful. Instead of making a standard CRUD-style app, I explored how a digital product could preserve memories in a way that feels warm, personal, and visually special.
